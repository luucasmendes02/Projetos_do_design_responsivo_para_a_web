Server specifications

A server is a computer equipped with one or more processors, memory banks, communication ports, and occasionally some system for storing data such as internal hard disks or SSD memories. Capable of running a specific set of programs or protocols to provide services to other machines or clients, servers are equipment dedicated to running applications and services within a LAN or WAN network.

How to choose a good server

To choose a server, you must be aware of the exact number and type of applications that will run, in addition to the number of users who will be connected. Large infrastructures often use the virtualization technique that has been increasingly adopted as a possible low-cost solution to meet basic information security requirements: Integrity, Confidentiality, and Availability.


  • To better describe them, we will present the characteristics of 3 levels of servers.

    • Basic, medium and advanced.



Basic server

It is the perfect server type for small applications. The device has reduced cost, and dispenses with air-conditioned environment. In addition, it reduces the rate of intrusions by centralizing information in one place.
It can be used for web applications like Apache, ISS, Nginx. Supports database such as MYSQL, Postgres. Ideal for simultaneous use of up to 20 customers in midsize applications.


  • Specifications:

    • Processor: Intel® Frequency: 3.6 GHz, Cache: 8 MB, Cores: 4 Segments: 8
    Memory: 12 Gb - DDR4-2666
    Intel® Virtualization Technology (VT-x): Yes
    Hard drive: 3.5" wired SATA, 6 Gbit/s, 7,200 RPM, 1 TB
    Network Card: 1 Gbit
    Power supply: 300 W wired
    Operating system support without factory installation: VMware ESXi, Windows Server® 2012, Standard, Red Hat Enterprise.



Medium server

This is a server for larger applications. This is a rack server needs to be disposed of in an air-conditioned environment.
It can be used for web applications like Apache, ISS, Nginx. Supports database such as MYSQL, Postgres, SQL Server. Ideal for simultaneous use of up to 60 clients in mid-sized intranet applications.


  • Specifications:

    • Processor: Intel® Frequency: 2.1 GHz, Cache: 11 MB, Cores: 8 Segments: 16
    Memory: 16 Gb - DDR4-2666
    Intel® Virtualization Technology (VT-x): Yes
    Hard drive: SATA , 6 Gbit/s, 7,200 RPM, 1 TB – Hot-plug Hard Drive
    Network Card: 1 Gbit (2und)
    Power supply: Single (1+0) automatic connection (550W)
    Operating system support without factory installation: VMware ESXi, Windows Server® 2012, Standard, Red Hat Enterprise.



Advanced server

This is a server for advanced applications. This is a rack server needs to be disposed of in an air-conditioned environment.
It can be used for web applications like Apache, ISS, Nginx. Supports database such as MYSQL, Postgres, SQL Server, Oracle. Ideal for high simultaneous customer utilization.


  • Specifications:

    • Processor: Intel® Frequency: 2.1 GHz, Cache: 16 MB, Cores: 12 Segments: 24
    Memory: 32 Gb - DDR4-2666
    Intel® Virtualization Technology (VT-x): Yes
    Hard drive: SATA , 6 Gbit/s, 7,200 RPM, 1 TB – Hot-plug Hard Drive
    Network Card: 1 Gbit (6und)
    Power supply: With redundancy (1+1) automatic connection (550W)
    Operating system support without factory installation: VMware ESXi, Windows Server® 2012, Standard, Red Hat Enterprise.
